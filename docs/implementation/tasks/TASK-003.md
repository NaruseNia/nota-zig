# TASK-003: 先読み・非貪欲量指定子の正規表現拡張

- **関連要件**: REQ-REGEX-007, REQ-REGEX-008
- **規模**: M
- **依存タスク**: TASK-002

## 実装概要

正規表現パーサーに先読み (`(?=...)`, `(?!...)`) と非貪欲量指定子 (`*?`, `+?`, `??`) のサポートを追加する。

## 対象ファイル

- `src/regex/ast.zig`
- `src/regex/parser.zig`

## 実装ステップ

1. AST に `Lookahead` ノード（positive/negative）を追加
2. パーサーに `(?=...)` と `(?!...)` の構文解析を追加
3. AST の `Repeat` ノードに `greedy: bool` フラグを追加
4. パーサーに `*?`, `+?`, `??` の解析を追加
5. 先読みと非貪欲のテストを作成
