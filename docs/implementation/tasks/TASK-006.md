# TASK-006: 先読みの DFA 統合

- **関連要件**: REQ-REGEX-007
- **規模**: M
- **依存タスク**: TASK-003, TASK-005

## 実装概要

先読み (`(?=...)`, `(?!...)`) を DFA ベースのエンジンに統合する。先読みは DFA 単体では表現できないため、補助 DFA + 後処理で実装する。

## 対象ファイル

- `src/regex/nfa.zig`
- `src/regex/dfa.zig`
- `src/engine/compiler.zig`

## 実装ステップ

1. 先読みパターンを独立した DFA としてコンパイル
2. メイン DFA の受理状態に先読み条件を関連付け
3. ランタイムで受理時に先読み DFA を実行して最終判定
4. 否定先読みの場合は先読み DFA が不受理なら全体として受理
5. 先読みを含むパターンのテストを作成
