# TASK-011: ファズテスト

- **関連要件**: REQ-INFRA-005
- **規模**: S
- **依存タスク**: TASK-008

## 実装概要

Zig の built-in fuzz testing を使い、任意入力で Lexer がクラッシュしないことを検証する。

## 対象ファイル

- `src/fuzz.zig` — ファズテストエントリポイント

## 実装ステップ

1. ファズテスト用の Lexer を定義（一般的なトークンセット）
2. 任意のバイト列を入力として全トークンを消費するテストを作成
3. パニックやメモリ安全違反が発生しないことを確認
4. `build.zig` にファズテストステップを追加
5. `mise run fuzz` で実行できることを確認
6. 短時間実行（1分）と長時間実行（1時間）のモードを用意
