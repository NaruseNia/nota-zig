# TASK-002: 正規表現パーサー

- **関連要件**: REQ-REGEX-001, REQ-REGEX-004, REQ-REGEX-005, REQ-REGEX-006
- **規模**: L
- **依存タスク**: TASK-001

## 実装概要

正規表現文字列を comptime で AST に変換するパーサーを実装する。基本セット（文字クラス、量指定子、選択、グループ）を対象とする。

## 対象ファイル

- `src/regex/ast.zig`
- `src/regex/parser.zig`

## 実装ステップ

1. `ast.zig` に Regex AST ノード型を定義（Literal, CharClass, Dot, Concat, Alt, Repeat, Group）
2. `parser.zig` に comptime パーサーを実装
   - 文字リテラルとエスケープシーケンス (`\w`, `\d`, `\s`, `\t`, `\n`, `\r`, `\\`)
   - 文字クラス `[...]`, `[^...]`, 範囲 `a-z`
   - 量指定子 `*`, `+`, `?`, `{n}`, `{n,}`, `{n,m}`
   - 選択 `|`
   - グループ `(...)`
   - `.` (任意バイト)
3. 演算子の優先順位を正しく処理（連結 > 選択）
4. 不正な正規表現に対するコンパイルエラーメッセージを実装
5. 各構文要素のテストを作成
