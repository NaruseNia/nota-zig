# REQ-PERF: 性能・ベンチマーク

## REQ-PERF-001: 手書き Lexer 同等以上の性能

- **優先度**: Must
- **ステータス**: Draft
- **説明**: comptime DFA のテーブルドリブン実行により、手書き switch-case Lexer と同等以上のスループットを実現する。プロジェクトのセールスポイント。
- **受入条件**:
  - [ ] JSON Lexer ベンチマークで手書き実装の 90% 以上のスループット
  - [ ] C-like 言語 Lexer ベンチマークで手書き実装の 90% 以上のスループット
  - [ ] ランタイムでのヒープアロケーションがゼロ
  - [ ] DFA テーブルがキャッシュフレンドリーなレイアウトで生成される
- **依存要件**: REQ-REGEX-003
